# Inventory-Management-JS
# Inventory Management System using JS

A clean, responsive, and fully client-side **inventory management web app** built with **HTML**, **CSS**, and **JavaScript**.  
Perfect for small businesses, warehouse staff, or personal stock tracking — no backend or database required. All data persists in the browser’s `localStorage`.

## Live Demo & Features

- **Add new products** – name, quantity, price, and category.
- **Edit & delete** existing items dynamically.
- **Search & filter** by product name or category.
- **Low stock alerts** – visual warning when quantity ≤ 5.
- **Real-time statistics** – total products, total value, low stock count.
- **Responsive design** – works on desktop, tablet, and mobile.
- **Data persistence** – inventory is saved automatically in your browser.

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML5 | Structure & semantic layout |
| CSS3 | Modern styling (Flexbox, Grid, custom properties) |
| JavaScript (ES6+) | DOM manipulation, event handling, localStorage, dynamic rendering |
